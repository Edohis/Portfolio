La présente politique de confidentialité contient toutes les informations et explications sur les données que nous sommes susceptibles de stocker concernant votre navigation si vous avez accepté que nous les stockions.

Si vous avez des questions supplémentaires ou si vous souhaitez obtenir plus d'informations sur notre politique de confidentialité, n'hésitez pas à nous contacter.

Consentement

En utilisant notre site web, vous consentez par la présente à notre politique de confidentialité et acceptez ses conditions.

Quelles données sont stockées

Toutes les données stockées sur votre navigateur ne sont utilisées que sur ce site. Ces données comprennent les données relatives à la langue dans laquelle le site est affiché ainsi que le thème dans lequel le site est affiché. En dehors de ces deux types de données, nous ne stockons aucun autre type de données.

Toutefois, il est possible que certaines des bibliothèques que nous utilisons sur ce site stockent elles-mêmes des données sur votre navigateur. Pour plus d'informations, nous avons listé ci-dessous sur cette page toutes les bibliothèques et autres éléments utilisés pour créer ce site.

Comment nous utilisons vos informations

Bien que nous stockions des données sur votre navigateur, ce site ne collecte aucun type de données. Les données stockées sont uniquement utilisées pour déterminer la langue dans laquelle ce site doit être affiché et le thème du site. En outre, ces données peuvent être supprimées par vous à tout moment. Si vous voulez savoir comment procéder, nous vous l'expliquons ci-dessous.

Cookies et pixels espions

Comme tout autre site web, ce site utilise des "cookies" (session et stockage local). Ces cookies sont utilisés pour stocker des informations, notamment les préférences des visiteurs et les pages du site Web auxquelles le visiteur a accédé ou qu'il a visitées. Ces informations sont utilisées pour optimiser l'expérience des utilisateurs en personnalisant le contenu de nos pages web en fonction du type de navigateur des visiteurs et/ou d'autres informations.

Vous pouvez choisir de désactiver les cookies par le biais des options de votre navigateur individuel. Pour connaître des informations plus détaillées sur la gestion des cookies avec des navigateurs web spécifiques, vous pouvez consulter les sites web respectifs de ces navigateurs.
